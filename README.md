# °coolWIRE v2.0

Kälteplanungstool von coolsulting e.U.

## Start
```
START_coolWIRE.bat
```
oder:
```
python -m streamlit run coolWIRE_main.py --server.port 8580
```

## Struktur
- `coolWIRE_main.py` – Hauptapp
- `modules/` – Kabelmatrix, Kühlstellen, KI-Import, Datenbank
- `.streamlit/secrets.toml` – API-Keys (NICHT ins Repo!)

## Dependencies
```
pip install -r requirements.txt
```
